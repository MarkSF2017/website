Plan de realizare a proiectului

Etapa 1
Studiu tehnic - data finalizare : 10.12.2014

Raport etapa 1 (decembrie 2014)
  1. 1.1 Studiul si analiza metodei de potabilizare a apei prin osmoza inversa.
  2. 1.2 Studiul si analiza solutiilor de denitrificare a apei reziduale.
  3. 1.3 Studiul si analiza proceselor auxiliare osmozei inverse.
  4. 1.4 Elaborarea preliminara a metodei de filtrare prin RO.

Etapa 2
Verificarea preliminara a solutiilor de filtrare. Stabilirea metodei finale de filtrare prin RO.
Elaborarea documentatiei modelului functional. - data finalizare : 31.07.2015

  1. 2.1.Verificarea experimentala a metodei de filtrare prin RO cu presiune ultrascazuta. Stabilirea metodei finale de filtrare prin RO.
  2. 2.2.Verificarea experimentala a solutiei propuse de denitrificare utilizand celule microbiene de electroliza.
  3. 2.3.Elaborarea si verificarea experimentala a solutiilor propuse de pompare si filtrare inainte si dupa procesul de osmoza inversa.
  4. 2.4.Elaborare documentaţie model funcţional – modul RO.
  5. 2.5.Proiectare model funcţional – instalaţie.
  6. 2.6.Elaborare documentaţie model funcţional – Verificare tehnologică.

Etapa 3
Realizare model funcţional - data finalizare : 10.12.2015

  1. 3.1.Realizare model funcţional – optimizare energetică.
  2. 3.2.Realizare model funcţional-montaj, reglaje, control modul denitrificare.
  3. 3.3.Realizare model functional – instalatie.
  4. 3.4.Realizare model funcţional (subansamble şi repere componente).
  5. 3.6.Diseminarea rezultatelor; participarea la manifestari tehnico-stiintifice; vizite de lucru/schimburi de buna practica.

Etapa 4
Experimentare şi demonstrare utilitate model. Diseminarea rezultatelor şi protejarea drepturilor de proprietate intelectuală - data finalizare : 30.06.2016

  1. 4.1.Experimentare modul– osmoză inversă.
  2. 4.2.Experimentare modul de denitrificare.
  3. 4.3.Experimentare instalaţie model funcţional.
  4. 4.4.Experimentare model funcţional – subansamble.
  5. 4.5.Demonstrarea funcţionalităţii şi utilităţii modelului.
  6. 4.6.Diseminarea rezultatelor. Participarea la manifestări tehnico-ştiintifice. Vizite de lucru/schimburi de bună practică.
  7. 4.7.Protejarea drepturilor de proprietate intelectuala (brevetarea solutiilor).
